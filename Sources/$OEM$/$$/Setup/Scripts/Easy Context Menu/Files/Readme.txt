

Se você adicionar qualquer arquivo na pasta Arquivos do Ec Menu v1.5, ele será automaticamente copiado para o seguinte caminho:
%ProgramFiles%\Menu de Contexto Fácil\Arquivos\
depois de adicionar qualquer aplicativo portátil nesta pasta, você pode adicioná-lo à lista do menu Ec usando o Editor de listas
nircmd é apenas um exemplo, é claro que podemos adicionar qualquer aplicativo arrastando e soltando na interface do menu de contexto fácil 
mas se fizermos isso, o programa portátil não funcionará no pendrive, é por isso que devemos adicionar o programa
na pasta de arquivos primeiro

Items.ini: contém a lista da interface do menu de contexto
EcMenu.ini: contém alguns arquivos de configuração e idioma

If you add any file into Ec Menu v1.5 Files folder it will be automatically copied to the following path:
%ProgramFiles%\Easy Context Menu\Files\
after adding any portable Application in this folder you can add it in to the Ec menu list with using List Editor
nircmd is just an exapmle , of course we can add any application with drag and drop it on the Easy context menu interface 
but if we do that portable program will not work in the Usb stick that is the reason why we should add the program
in File folder firs

Items.ini : contain context menu interface's list
EcMenu.ini : contain some setting and Language files
